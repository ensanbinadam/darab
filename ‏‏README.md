## ✨ المميزات

- **الجدول الكامل**: ١٠٠ عملية مرتّبة من ١×١ حتى ١٠×١٠.
- **ترتيب الأسئلة**: اختيار بين العرض «📋 بالترتيب» أو «🎲 عشوائي»، ويُعاد الخلط تلقائياً عند كل محاولة جديدة في الوضع العشوائي.
- **طريقة التصحيح**:
  - «⚡ تدريب» — تصحيح فوري لكل إجابة مع إحصائية حيّة، مناسب للتعلّم.
  - «📝 اختبار» — يكتب الطالب كل الإجابات ويظهر التقييم عند الإنهاء، لقياس المستوى الحقيقي.
- **وضع المراجعة 👀**: يعرض النواتج لدراسة الجدول قبل الحل، ويحترم الترتيب المختار.
- **منطق رياضي صارم**: مقارنة عددية دقيقة، فلا تُقبل الأرقام المعكوسة (٨٢ ليست ٢٨)، ترسيخاً لمفهوم منزلة الأرقام.
- **دعم النظامين الرقميين**: يقبل الأرقام الإنجليزية (28) والهندية الشرق أوسطية (٢٨)، ويوحّد العرض على الأرقام الهندية أثناء الكتابة.
- **تنبيه تعليمي**: يذكّر الطالب أن الكتابة على الشاشة تبدأ من اليسار (العشرات أولاً ثم الآحاد).
- **شهادة تقدير 🏅**: تُمنح لمن حقّق ٩٠٪ فأكثر في وضع الاختبار، مع احتفالٍ بصريّ.
- **طباعة / حفظ PDF 🖨️**: إخراج تقرير النتيجة والشهادة بنفس الألوان والتصميم.
- **واجهة متجاوبة**: تعمل على الحاسب واللوح والجوال، باتجاه عربي صحيح (RTL).
- **ملفٌ واحد بلا تبعيّات**: كل شيء في صفحة HTML واحدة، تعمل دون إنترنت.

## 🚀 طريقة الاستخدام

**محلياً:** نزّل الملف `جدول_الضرب.html` وافتحه بأي متصفّح حديث (Chrome / Edge / Safari / Firefox). لا حاجة لأي تثبيت.

**النشر على الإنترنت عبر GitHub Pages:**

1. ارفع الملف إلى مستودعك على GitHub.
2. من إعدادات المستودع: `Settings` ← `Pages`.
3. اختر الفرع `main` والمجلد `/ (root)` ثم `Save`.
4. سيصبح المشروع متاحاً على رابطٍ مثل: `https://USERNAME.github.io/REPO/جدول_الضرب.html`

> نصيحة: يمكنك إعادة تسمية الملف إلى `index.html` ليُفتح المشروع مباشرةً عند زيارة رابط الصفحة.

### English Summary

**Multiplication Table Championship** — an interactive, single-file web app (HTML/CSS/vanilla JS, no dependencies) for learning and mastering multiplication tables from 1×1 to 10×10. Features ordered/random question modes, instant-feedback practice mode and graded exam mode, a review mode that reveals answers for study, strict numeric checking, support for both Western and Eastern-Arabic numerals, an achievement certificate (90%+) and print-to-PDF. Fully right-to-left Arabic UI, runs offline in any modern browser.
